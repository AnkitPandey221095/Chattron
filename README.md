# ✨ Chattron (MERN Project) ✨

A real-time full-stack chat application built using the MERN stack.
Chattron demonstrates modern web development concepts such as real-time communication, custom authentication, image sharing, and modern UI design.

## 🧰 Tech Stack & Dependencies

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.io
- JSON Web Tokens (JWT)
- Resend – Email service
- Cloudinary – Image storage
- Arcjet – API rate limiting

### Frontend
- React.js
- Tailwind CSS
- DaisyUI
- Zustand – State management

---

## 🎯 Features

- 🔐 **Custom JWT Authentication** – Secure signup, login, and logout system
- ⚡ **Real-time Messaging** – Instant chat powered by Socket.io
- 🟢 **Online/Offline Presence** – See which users are active
- 📨 **Welcome Emails** – Automatic email sent to new users
- 🗂️ **Image Sharing** – Upload and send images in chat using Cloudinary
- 🚦 **API Rate Limiting** – Prevents API abuse using Arcjet
- 🎨 **Modern UI** – Clean and responsive interface with Tailwind & DaisyUI
- 🧠 **State Management** – Efficient frontend state handling with Zustand

## � Wanna Clone and run this repo

### Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_REPOSITORY_URL>

# Step 2: Navigate to backend
cd backend

# Step 3: Install backend dependencies
npm install

# Step 4: Navigate to frontend
cd ../frontend

# Step 5: Install frontend dependencies
npm install

# Step 6: Configure environment variables
Create a .env file in both backend and frontend folders 
and add the required environment variables.

# Step 7: Run the application
npm run dev
```
